STEP 3 — ROOT CAUSE (Julia <-> Fortran source) — A WIRING BUG
--------------------------------------------------------------------------------
  Fortran (HydrologyNoDrainageMod.F90:339 -> SoilWaterPlantSinkMod.F90:18):
    Compute_EffecRootFrac_And_VertTranSink reads `use_hydrstress` from clm_varctl
    (module global). For the natural-veg (istsoil) column it branches:
      use_hydrstress=.true.  -> Compute_..._HydStress   (lines 124-126)
        qflx_rootsoi_col(c,j) = sum_p k_soil_root(p,j)*(smp(c,j)-vegwp(p,root)-grav2)*wtcol(p)
        (water-potential-gradient driven; distributes across all rooted layers)
      use_hydrstress=.false. -> Compute_..._Default     (line 136)
        qflx_rootsoi_col(c,j) = rootr_col(c,j) * qflx_tran_veg_col(c)
        (static transpiration-weighted root profile)

  Julia (src/biogeophys/soil_water_plant_sink.jl): the wrapper
    compute_effec_rootfrac_and_vert_tran_sink!(...; use_hydrstress=false)
    has use_hydrstress as a KEYWORD defaulting to false. BUT both driver call
    sites omitted it:
      src/driver/clm_driver.jl:1616          (forward driver, HydrologyNoDrainage)
      src/driver/driver_reverse.jl:292       (reverse driver rootsink phase)
    => in PHS mode (use_hydrstress=true) Julia still ran the DEFAULT method, so the
       sink followed the static rootr profile instead of the PHS k_soil_root/vegwp
       gradient. In this column the default rootr_col collapsed the sink onto the
       deepest layer (12) — hence dJ=0 in 2..11 and -0.0082 in 12.

  CLASSIFICATION: fixable wiring term (same class as the 4 prior PHS fixes), NOT a
  coupling-order/snapshot floor.

STEP 5 — VERDICT
--------------------------------------------------------------------------------
  The dominant H2OSOI_LIQ residual (the layer-12 root-sink collapse, J-F=-7.45e-3)
  was a FIXABLE WIRING BUG: use_hydrstress was not threaded into the soil-water
  root-sink call, so PHS mode used the static-rootr Default sink instead of the
  k_soil_root/vegwp HydStress sink. Fixed; layers 2..12 now bit-faithful (~1e-6).

  The remaining H2OSOI_LIQ residual is LAYER 1 ONLY (J-F = 1.44e-4 injected /
  surface). It is unchanged by this fix and is downstream of the surface
  soil-evaporation flux (qflx_ev_soil), which is set by the ground energy balance
  (T_GRND/T_SOISNO differ 4.4e-3 K). That is the energy-balance/T_VEG coupling-order
  snapshot floor (the Phase-3g carried hk_l -> bsha<0.01 PLC-clamp -> T_VEG/T_GRND
  chain), NOT a soil-water-process bug. So the soil-water axis is bit-faithful; the
  global residual reverts to the previously-classified canopy-time snapshot floor.
